人気記事のランキング機能を提供します。

== Description ==

このプラグインは人気がある記事をランキングします。
以下の機能を提供します。

* カスタム投稿タイプ対応
* 専用ウィジェット
* 専用ショートコード（ベータ）
* キャッシュ対応

**PHP5が使用できる環境が必要です**

== Installation ==

プラグインをダウンロード後、管理画面より有効化してください。

有効化後、"設定"の"ランキング"よりランキング条件の設定を行ってください。
ランキング条件の設定後、"ランキング"ウィジェットにて表示するランキング条件を指定できます。

ランキングは個別ページが表示される毎に1ポイント加点されていきます。
